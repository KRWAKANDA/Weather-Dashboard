Weather Dashboard Application
Project Overview

The Weather Dashboard Application is a frontend web application built using React and Tailwind CSS. The purpose of this project is to demonstrate the ability to consume third-party APIs, manage application state, and build a responsive user interface using modern frontend technologies.

This project forms part of my ALX Software Engineering portfolio, showcasing practical skills in React development, UI design, and API integration.

Problem Statement

Users often need quick and reliable access to weather information without navigating complex platforms. This application solves that problem by providing a simple interface where users can search for a city and instantly view current weather conditions.

Features

Search for real-time weather information by city name

Display temperature, weather condition, humidity, and wind speed

Responsive layout optimized for desktop and mobile screens

Clean and accessible UI built with Tailwind CSS

API-driven data fetching with error handling

Technologies Used

React – Component-based UI development

Tailwind CSS – Styling and responsive design

JavaScript (ES6+) – Application logic

Weather API – External data source for live weather information

Vite – Development and build tooling
