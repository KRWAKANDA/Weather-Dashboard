# Weather Dashboard

A React-based Weather Dashboard that allows users to search for current weather conditions in cities worldwide.  

## Features
- Search for cities
- Display temperature, humidity, wind speed, weather description, and icon
- Dynamic background based on weather
- Recent search history

## Tech Stack
- React
- Tailwind CSS v4
- OpenWeatherMap API

## Installation
1. Clone the repo: `(https://github.com/KRWAKANDA/Weather-Dashboard.git)`
2. Install dependencies: `npm install`
3. Run locally: `npm start`
