# Weather App (React + WeatherAPI)

A modern React application that provides real-time weather information using the WeatherAPI service. Users can search for any city worldwide and view current weather conditions, including temperature, humidity, wind speed, and weather descriptions.

## Features

- Search weather by city name
- Real-time current weather data
- Displays temperature, humidity, wind speed, and conditions
- Responsive and user-friendly UI
- Error handling for invalid or unknown locations
- Loading states for improved UX

## Tech Stack

- **Frontend:** React (JavaScript)
- **API:** WeatherAPI
- **HTTP Client:** Fetch API / Axios
- **Styling:** CSS / Tailwind CSS 
- **Build Tool:** Vite / Create React App

## Prerequisites

Ensure you have the following installed:

- Node.js (v16 or higher recommended)
- npm or yarn
- A WeatherAPI account and API key
